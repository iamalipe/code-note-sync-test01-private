Safal Global Auth System

Feature  
- User Management for Admin  
- User Management for Business  
- Login, Register, Logout API  
- Change Log System

Roles - Super Admin - Can do anything  
Business - Can add more user  
User - Can change only there info (Username, email, password)

Require  
- Seperate DB  
- Seperate Server  
- Seperate Domain (Mobile device need more time to setup)

Web UI Technology  
React, Typescript, Ant Design, Tailwind

Backend Technology  
Express, NodeJs, Typescript, MongoDB, Prisma ORM

Time Require  
Backend - 3 days/24h (approximate)  
Web UI - 2 days/16h (approximate)

Existing Application changes take more time, because it currenly linked to that application database.  
if we need to add this system to our Existing Application, then we need to wipe the application users database.  
Clone is also possible to Auth System DB.
